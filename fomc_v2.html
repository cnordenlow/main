Methodology
The approach of this project are as follows.

  • Words and topics are divided into bag of words of its meaning (e.g. the word “strong” may consists of [“strong”, “stronger”, “strongest”, “strengthen”]).

  • All text are divided into sentences that cleaned for special characters, extra spaces, stop words and then tokenized and lemmatized, then converted back to it´s sentence.

  • All sentences are then iterated where it checks for the different bag of words and then counted, (e.g. if the word “stronger” is found in a sentence, the count adds one.) For some topics, a second loop are then iterated where it counts negative and positive words in the same meaning to get the net sentence.

The methodology used in this project include use of Python for web parsing and the package of NLTK for text mining. The data is then transfered to R for some additional calculations and to provide vizualisations in R Markdown. Some notes below of the different parts.

Web parsing
All web parsing is done using BEautifulSoup package. THe code first parse all .htm pages with the url of “Minutes” in it. Then, a second loop parse all the (chosen) Minutes for the text mining. In both parsing steps, there are a time delay of a couple of seconds.
